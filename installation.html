Installation

Install with Pip

Note the name is virga-exo (with an -exo at the end).

pip install virga-exo

Install with Git

git clone https://github.com/natashabatalha/virga.git
cd virga
python setup.py install

Download the Refractive Index Files

Virga v2 includes the ability to do aggregates. This requires a different database of optical properties if you are interested in those. Otherwise you are sitll okay to use the spherical particle only database. For climate modeling (paired with picaso) spherical particle database is recommended.

  • Spherical: Download and read about these for spherical particles (either v1 or v2).

  • Aggregates: Download and read about these for aggregate particles (only for v2).
